# HTML Practice: Assignment 2

This folder contains the second assignment for HTML practice, focusing on presenting information about a specific topic (an informational page about a cat) and introducing basic visual styling.

## Technologies Used
*   **HTML**: For structuring the informational content.
*   **CSS (Cascading Style Sheets)**: Introduced at a basic level to provide visual styling to the raw HTML structure.

## Core Concepts Covered
*   **Content Structuring**: Combining headings, paragraphs, and potentially media elements to create a cohesive, readable document layout.
*   **External vs Internal Styling**: Understanding how to apply basic CSS rules to modify the default browser rendering of HTML elements, improving the presentation of the content.
*   **Semantic Meaning**: Ensuring that the tags chosen (like `<strong>` for importance rather than just `<b>` for bolding) convey the correct meaning to both the browser and assistive technologies.

## How to Run
Open `aboutCat.html` directly in any web browser to view the page.
