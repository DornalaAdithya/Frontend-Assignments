# HTML Practice: Assignment 4

This folder contains the fourth assignment, focusing on creating more complex document structures such as business profiles and detailed test pages.

## Technologies Used
*   **HTML**: To create complex, content-heavy document structures.
*   **CSS**: To provide custom styling rules across multiple HTML files.

## Core Concepts Covered
*   **Complex Page Structure**: Managing larger amounts of HTML code and ensuring it remains organized, readable, and semantically correct.
*   **Data Representation**: Utilizing advanced HTML features (potentially tables `<table/>` or complex nested elements) to display structured data clearly to the user.
*   **CSS Selectors and Specificity**: Applying specific CSS rules to target particular sections of the complex HTML structure without unintentionally affecting other areas of the page.

## How to Run
Open either `bussinessPage.html` or `testPage.html` directly in any web browser to view the respective pages.
