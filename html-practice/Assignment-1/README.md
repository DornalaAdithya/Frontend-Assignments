# HTML Practice: Assignment 1

This folder contains the first assignment for HTML practice, focusing on the fundamental building blocks of a webpage and organizing text data.

## Technologies Used
*   **HTML (HyperText Markup Language)**: The foundational markup language used to structure content on the web.

## Core Concepts Covered
*   **Document Structure**: Understanding the basic skeleton of an HTML document, including the `<!DOCTYPE html>` declaration, the `<html>` root element, the `<head>` for metadata, and the `<body>` for visible content.
*   **Semantic Text Elements**: Using tags like `<p>` (paragraphs), `<h1>` through `<h6>` (headings) to give semantic meaning to text.
*   **HTML Lists**: Organizing information into structured formats.
    *   **Unordered Lists (`<ul>`)**: Used for collections of items where the order does not matter (typically rendered with bullet points).
    *   **Ordered Lists (`<ol>`)**: Used for collections of items where the sequence is important (typically rendered with numbers or letters).
    *   **List Items (`<li>`)**: The individual elements contained within `<ul>` or `<ol>` tags.
*   **Nested Lists**: Placing a list inside another list item to create hierarchical data structures.

## How to Run
Open `simpleWebPage.html` or `nestedLists.html` directly in any web browser to view the rendered pages.
