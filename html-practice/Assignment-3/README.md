# HTML Practice: Assignment 3

This folder contains the third assignment, which involves building a structured restaurant informational page.

## Technologies Used
*   **HTML**: To define the core layout and sections of the restaurant page.
*   **CSS**: To enhance the visual appeal and layout separation.

## Core Concepts Covered
*   **Sectioning Elements**: Using semantic tags (such as `<header>`, `<footer>`, `<section>`, or `<article>`) or structural `<div>` elements to break the webpage down into logical, manageable parts (e.g., separating the menu from the contact information).
*   **Media Elements**: Incorporating images (`<img>`) to enhance the visual storytelling of the page, along with understanding attributes like `src` (source) and `alt` (alternative text for accessibility).
*   **Inline and Block Elements**: Deepening the understanding of how different HTML elements behave in the document flow (e.g., how a `<span>` sits within text versus how a `<div>` forces a new line).

## How to Run
Open `restaurantPage.html` directly in any web browser to view the page.
