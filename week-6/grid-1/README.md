# CSS Grid Layout: Assignment 1

This folder contains the first assignment focusing on the CSS Grid Layout Module. It demonstrates fundamental grid concepts for structuring web pages.

## Technologies Used
*   **HTML (HyperText Markup Language)**: The standard markup language for creating web pages. It defines the structure and content of the document using elements like `<div>`, `<header>`, and `<section>`.
*   **CSS (Cascading Style Sheets)**: A style sheet language used for describing the presentation of a document written in HTML. It controls the layout, colors, and fonts.

## Core Concepts Covered
*   **CSS Grid Module**: A two-dimensional layout system for the web. It lets you lay content out in rows and columns, offering fine-grained control over the alignment and sizing of elements.
*   **Grid Container**: The parent element on which `display: grid` is applied. All direct children of this container become grid items.
*   **Grid Template Columns & Rows**: Properties used to define the track sizing (the width of columns and height of rows) in the grid layout.
*   **Grid Gap**: Properties used to specify the size of the grid lines (the empty space between columns and rows).

## How to Run
Simply open the `index.html` file in any modern web browser to view the layout.
