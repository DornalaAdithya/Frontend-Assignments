# Frontend Hands-on: Assignment 1

This folder contains the first hands-on assignment for week 6, focusing on building a structured web page using core HTML and CSS techniques.

## Technologies Used
*   **HTML**: Used to structure the semantic meaning of the content (headings, paragraphs, lists).
*   **CSS**: Used to control the visual presentation, spacing, and colors of the HTML elements.

## Core Concepts Covered
*   **The Box Model**: A fundamental concept in CSS where every element is represented as a rectangular box. It consists of margins, borders, padding, and the actual content. Understanding this is crucial for layout and spacing.
*   **Element Selectors**: CSS rules applied to specific HTML tags, classes (`.classname`), or IDs (`#idname`) to target them for styling.
*   **Document Flow**: The default way browsers position elements on a page, determining how block-level and inline elements stack and behave.

## How to Run
Simply open the `index.html` file in any modern web browser to view the result.
