# CSS Grid Layout: Assignment 2

This folder contains the second assignment focusing on advanced implementations of the CSS Grid Layout Module.

## Technologies Used
*   **HTML**: Provides the semantic structure for the web page content.
*   **CSS**: Applies styling and complex grid rules to structure the user interface.

## Core Concepts Covered
*   **CSS Grid Areas**: A technique that allows you to name specific grid items and position them by referencing those names in a template, making complex layouts highly readable in the CSS code.
*   **Fractional Units (`fr`)**: A flexible length unit specific to CSS Grid that represents a fraction of the available space in the grid container.
*   **Responsive Grid**: Utilizing grid properties alongside media queries to create layouts that adapt fluidly to different screen sizes.

## How to Run
Simply open the `index.html` file in any modern web browser to view the layout.
