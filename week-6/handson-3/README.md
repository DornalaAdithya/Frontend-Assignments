# Frontend Hands-on: Assignment 3

This folder contains the third hands-on assignment for week 6, focusing on responsive design and modern layout structures.

## Technologies Used
*   **HTML**: Used to build the skeleton of the webpage.
*   **CSS**: Used to style the webpage and implement responsive layout logic.

## Core Concepts Covered
*   **Responsive Web Design (RWD)**: An approach to web design that makes web pages render well on a variety of devices and window or screen sizes from minimum to maximum display size.
*   **Media Queries**: A CSS technique introduced in CSS3 that changes the presentation of content to a specific range of output devices without changing the content itself (e.g., `@media (max-width: 768px)`).
*   **Fluid Layouts**: Using relative units (like percentages `%` or viewport units `vw`, `vh`) instead of fixed units (like `px`) to allow elements to scale seamlessly.

## How to Run
Simply open the `index.html` file in any modern web browser to view the result. Resize the browser window to observe how the layout adapts.
