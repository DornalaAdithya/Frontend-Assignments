# Frontend Hands-on: Assignment 2.2

This folder contains part 2 of the second hands-on assignment for week 6, building upon layout techniques with additional styling.

## Technologies Used
*   **HTML**: Defines the structural elements of the layout.
*   **CSS**: Handles the visual presentation and exact positioning.

## Core Concepts Covered
*   **Flexbox (Flexible Box Layout)**: A one-dimensional layout method for arranging items in rows or columns. It provides a more efficient way to lay out, align, and distribute space among items in a container, even when their size is unknown and/or dynamic.
*   **Flex Container**: An element on which `display: flex` is set.
*   **Flex Properties**: Techniques such as `justify-content` (aligning items along the main axis) and `align-items` (aligning items along the cross axis).

## How to Run
Simply open the `index.html` file in any modern web browser to view the result.
